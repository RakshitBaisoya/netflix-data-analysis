# Netflix Data Analysis 📊

## 📌 Project Overview
This project analyzes Netflix's content dataset to extract insights about movies and TV shows available on the platform.

## 🎯 Objective
To understand content distribution, trends over time, and key patterns in Netflix data.

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib
- Seaborn

## 📂 Dataset
Netflix Movies and TV Shows dataset (Kaggle)

## 🔍 Key Analysis Performed

### 1. Movies vs TV Shows
- Compared the number of movies and TV shows available on Netflix.
- Found that movies dominate the platform.

### 2. Top Content Producing Countries
- Identified top 10 countries contributing content.
- USA and India are among the top contributors.

### 3. Content Growth Over Years
- Analyzed how content was added over time.
- Significant growth observed after 2015.

### 4. Content Ratings
- Identified most common ratings.
- TV-MA and TV-14 are the most frequent.

## 📈 Key Insights
- Netflix has more movies than TV shows.
- Content addition increased rapidly in recent years.
- Certain countries dominate content production.
- Mature content ratings are more common.

## 🚀 Conclusion
This project demonstrates data cleaning, analysis, and visualization skills using Python. It highlights important trends in Netflix's content strategy.
